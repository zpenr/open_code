# SPECIFICATION

Разработать Python-модуль, который читает текстовый файл с данными о продажах, выполняет простые расчеты и формирует текстовый отчет.

## Формат входных данных

Входной файл — UTF-8 текст, где каждая корректная строка имеет вид:

`product_name,category,unit_price,quantity`

Пример:

`tea,drinks,7.0,5`

## Правила обработки

- строка должна содержать ровно 4 поля;
- `unit_price` должен преобразовываться в число;
- `quantity` должен преобразовываться в целое число;
- некорректные строки не должны останавливать обработку;
- некорректные строки должны игнорироваться.

## Требуемые функции модуля

### 1. Чтение данных

Публичная функция должна:

- принимать путь к файлу;
- читать файл построчно;
- преобразовывать корректные строки во внутренние записи;
- возвращать список записей.

### 2. Вычисление общей суммы

Публичная функция должна:

- принимать список записей;
- вычислять сумму `unit_price * quantity` по всем записям;
- поддерживать необязательную скидку в процентах;
- возвращать итоговую сумму.

### 3. Поиск по порогу

Публичная функция должна:

- принимать список записей и порог;
- возвращать только те записи, у которых сумма по позиции не меньше порога.

### 4. Агрегация по категориям

Публичная функция должна:

- принимать список записей;
- возвращать отображение вида:

`category -> total_amount`

### 5. Формирование отчета

Публичная функция должна:

- принимать список записей;
- включать суммы по категориям;
- включать итоговую общую сумму;
- возвращать многострочный текст.

Пример структуры отчета:

```text
Report
------
drinks: 72.5
bakery: 55.6
fruit: 24.0
------
Total: 152.1
